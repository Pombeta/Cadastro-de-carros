# 🚗 Sistema de Cadastro de Veículos - Concessionária

Este projeto é um sistema simples desenvolvido em **Java** para simular o cadastro de veículos em uma concessionária.

O programa roda no **terminal** e permite cadastrar veículos, listar os carros cadastrados e calcular uma projeção de vendas.

## 📋 Funcionalidades

O sistema possui as seguintes opções:

1 - Cadastrar veículo  
2 - Listar veículos cadastrados  
3 - Projeção de vendas  
0 - Sair do sistema  

### Cadastro de veículos
O usuário pode cadastrar:

- Nome do carro
- Cor do veículo
- Placa
- Valor de venda

### Listagem de veículos
Mostra todos os veículos cadastrados com:

- Nome
- Cor
- Placa
- Valor de venda

### Projeção de vendas
Calcula:

- Valor total das vendas
- Comissão de 20% sobre o total

## 🛠 Tecnologias utilizadas

- Java
- ArrayList
- Estrutura de repetição (while)
- Estrutura condicional (switch)
- Métodos para organização do código

Este projeto foi desenvolvido para praticar conceitos fundamentais de programação em **Java**, como:

- Organização do código com métodos
